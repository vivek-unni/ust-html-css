Container Tags

Container Tags - Tags that enclose other content. Examples include:
  • <ul> - Unordered List
  • <table> - Table
  • <body> - Body
  • <head> - Head
  • <div> - Division
 Standalone Tags - Tags that do not require a closing tag. Examples include:
  • <br> - Line Break
  • <img> - Image
  • <hr> - Horizontal Rule
 Content Tags - Tags that must be opened and closed. Examples include:
  • <h1> - Heading 1
  • <td> - Table Data
  • <p> - Paragraph
  • <b> - Bold
  • <i> - Italics
 Parent and Child Tags - Tags that can contain other tags. Examples include:
  • <table> - Contains <tr> and <td>
  • <ul> - Contains <li>
 Tags with Attributes - Tags that can have additional attributes. Examples include:
  • <div id="myId">
  • <span class="myClass">
  • <img src="image.jpg" alt="description">
